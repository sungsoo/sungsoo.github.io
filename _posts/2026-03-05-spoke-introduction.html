3. SPOKE 엔진

Live Database

65개 이상의 전문 DB 통합,
1억 5천만 개 이상의 관계

SPOKE는 속성 그래프(Property Graph) 구조를 통해 새로운 데이터가 추가될 때마다 연결망이 확장되는 프랙탈 구조를 가집니다.

9,000+ 질병 약물 표적 & 부작용 GWAS 유전자 데이터 30만 종 미생물
🦠

미생물 & 엑톰

병원체와 공생균의 기전 분석은 물론, Zip code 기반의 수질/독성 노출 데이터까지 연결하여 질병의 근본 원인을 추적합니다.

5. AI와 대화하는 지식 그래프

R

RAG (검색 증강 생성)

LLM이 SPOKE의 데이터를 실시간 참조하여 환각 현상을 방지하고 정확한 근거를 제시합니다.

M

AI가 데이터베이스에 직접 접속하여 복잡한 통계 분석 및 추론을 수행합니다.

맥락화된 지식이 여는
새로운 의학의 시대

SPOKE 지식 그래프는 단순한 데이터 집합체가 아닌, 생물학적 인과관계의 경로를 명확히 하는 '살아있는 생명의 지도'입니다. 데이터의 파편화를 극복하고 통합된 관점에서 환자를 바라보는 미래를 그립니다.
